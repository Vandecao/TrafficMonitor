Welcome to the TrafficMonitor wiki!

TrafficMonitor is a suspension window software for Windows which can be used for network speed monitoring.

Here, you can learn about the functions and detailed usage of TrafficMonitor.

The main function of TrafficMonitor is to display the current network speed, CPU and memory usage of the computer. The main interface includes a main window (desktop suspension window) and a taskbar window.
