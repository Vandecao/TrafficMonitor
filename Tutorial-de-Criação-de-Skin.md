A partir da versão 1.00, o TrafficMonitor adicionou temas no formato XML. Ele continua compatível com os temas no formato INI das versões anteriores.

Este artigo apresenta principalmente como criar temas em formato XML. Se você deseja ver como criar temas para versões mais antigas, consulte o " [Tutorial sobre como criar temas para versões antigas](https://github.com/Vandecao/TrafficMonitor/wiki/xxxxxxxx) ".

Etapas de criação da skin
Os arquivos de skin do TrafficMonitor estão localizados no mesmo skinsdiretório do programa principal. Cada skin é colocada em uma pasta separada, e o nome da pasta é o nome da skin.

Para criar um novo tema, você precisa criar uma pasta com o nome do tema e, em seguida, colocar o produto final background.bmp e background_l.bmp duas imagens de fundo nessa pasta.

Crie um skin.xml arquivo de configuração chamado " ", codificado em UTF8. Edite este arquivo de configuração para configurar os itens de exibição incluídos no tema.

Consulte a próxima seção para obter mais skin.xml explicações.
